性能测试页面

这是一个独立的路由页面，用于测试路由切换时的性能采集

Web Vitals 采集说明

  • FCP, TTFB, LCP - 页面加载时自动上报
  • FID - 点击下方按钮触发首次交互
  • CLS, INP - 关闭此页面时触发页面隐藏事件

测试步骤

  1. 打开 Network 面板，筛选 /batch 请求
  2. 点击下方"触发 FID"按钮触发首次交互
  3. 点击"关闭页面"按钮触发 CLS 和 INP 上报
  4. 查看 Network 面板的 /batch 请求，验证 Web Vitals 事件

预期结果

关闭页面后，打开 Network 面板查看最新的 /batch 请求，应该包含：

  • type: "webVital", name: "FCP"
  • type: "webVital", name: "TTFB"
  • type: "webVital", name: "LCP"
  • type: "webVital", name: "FID"（如果点击了按钮）
  • type: "webVital", name: "CLS"（关闭页面时上报）
  • type: "webVital", name: "INP"（关闭页面时上报）
